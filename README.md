# nodejs-cli-crypto-bcrypt-encoded

## Description
A bases for encoding a password
with bcrypt.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
